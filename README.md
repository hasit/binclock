binclock
========

## Description ##
A simple C++ program to show system time with binary clock

## How to run ##
```
$ git clone https://github.com/hasit/binclock.git
```

```
$ cd ~/binclock/
```

```
$ g++ working_binclock.cpp -o wbc
```

```
$ ./wbc
```
